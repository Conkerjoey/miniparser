# Supported Syntax
### Values
- Decimal
- Octal
- Hexadecimal
- Binary
### Operators
The current supported operators are:
- addition (+)
- subtraction (-)
- multiplication (*)
- division (/)
- exponent (^)
- modulo (%)
### Variables
Variables name are supported. The current standard is any letters and the '_' character.

For example:
- x
- _abc
- x_x
- ABC
